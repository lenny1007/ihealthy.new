# 喝牛奶會拉肚子？什麼是乳糖不耐症？

乳糖不耐症是指消化乳糖的能力降低，乳糖是牛奶中的主要糖分。

大多數兒童和嬰兒會在小腸中產生乳糖酶，將乳糖分解為其可消化的糖成分（葡萄糖和半乳糖）。

然而，在全球大多數人口中，乳糖酶的產生在兒童時期急劇下降，並且由於成年後乳糖酶的產生減少或不產生，世界上約70%的成年人口存在一定程度的乳糖不耐症[^1]。

歐洲人有乳糖不耐症的人口比例最低，約為5-15%，而亞洲人則最高，約為90%[^2]。

在台灣，有很多人有乳糖不耐症，卻沒有意識到。

## ⚠️ 安全指南

::alert{type="warning"}
**重要提醒：**
- 乳糖不耐症不是疾病，是正常生理現象
- 避免乳製品時要確保鈣質和維生素D攝取
- 乳糖酶補充劑需在進食前服用

- 嚴重腹瀉時要補充水分，避免脫水
- 如有持續症狀，需排除其他腸胃疾病

正常情況下，乳糖分解酵素會將乳糖拆分為葡萄糖和半乳糖 —— 身體能夠輕鬆吸收的簡單糖。

但在乳糖不耐症的人身上，這種酵素正在休假，所以乳糖未經消化就直接通往結腸，在那裡細菌等著像參加宴會一樣地享用乳糖，引發各種胃腸道上的戲劇 —— 想像一下腹脹、氣體排出和腹瀉。

## 為何亞洲人群中乳糖不耐症更普遍？

在這裡，基因和飲食習慣都扮演了主要角色。

大多數亞洲人隨著年齡的增長，乳糖分解酵素活性會有所下降，這在長期從事牧牛並飲用未發酵牛奶的人群中較少看到。

## 症狀 — 你的腸道在說話

如果在享用富含乳製品的餐點後，你發現自己腹脹、打嗝或急衝衝跑向廁所，那麼你可能就有乳糖不耐症。

患有乳糖不耐症的人會因腸道細菌消耗未消化的乳糖而出現消化症狀[^3]。

症狀因人而異，可能包括脹氣和腹脹（由腸道細菌產生的過量氣體引起）、腸鳴（一個令人愉悅的詞，指的是因氣體和液體晃動而引起的肚子咕嚕聲）、腹瀉（由滲透壓引起）未消化的乳糖）、腹痛和噁心。症狀通常在攝取乳糖後 30 分鐘至 2 小時出現[^3]。

::alert{type="warning"}
如果你愛喝拿鐵，但總是在喝完後感到不舒服，那麼你可能就有乳糖不耐症。

## 飲食策略 — 拉不起來的褲子，我還躲不起嗎?

管理乳糖不耐症以一項原則為核心：避免症狀，同時還要確保為你的骨骼攝取足夠的鈣質[^4]。

少吃含有乳醣的食物，包括牛奶、奶酪、酸奶、冰淇淋和奶油[^4]。

策略包括逐漸適應乳糖消耗，選擇無乳糖商品，求助於乳糖分解酵素產品，以及從非乳製品的強項或補充劑中獲取鈣質[^4]。

有些人可以處理一點點乳糖 (最多大概一杯牛奶) ，特別是當它與餐食中的其他成份混合時[^5]。

## 乳糖不耐症的人可以訓練自己的腸胃道讓它們更能夠處理乳糖嗎?

從少量牛奶開始，慢慢增加數量，可以訓練你的肚子接受乳糖嗎?

::alert{type="danger"}
很遺憾的答案是不行的。 事情要是這麼簡單，那麼我們就不會有乳糖不耐症了[^6]。

## 乳糖不耐症有解嗎?

乳糖酶的產生量低甚至不產生實際上是成年人的正常現象[^7]；

它不是一種疾病，本身不需要治療[^7]。

繼發於疾病的乳糖不耐症通常是自限性的；當潛在的病症得到解決後，乳糖酶的產生通常會恢復，無需進一步治療[^7]。

在所有情況下，乳糖不耐症的消化症狀都可以透過以下三種方法之一來控制：

::list{type="primary"}
- **減少飲食中的乳糖直到症狀不再出現**
- **補充乳糖酶**
  - 有蠻多市售的乳糖分解酵素補充劑。
  - 當菜單上有乳製品時，吃一顆乳糖分解酵素補充劑。
- **補充益生菌**
  - 補充益生菌的效果比較有限，但是可以嘗試。
  - 建議直接選擇乳糖分解酵素補充劑。

## 你的乳糖不耐症生存 Tips

1. **發現你能夠容忍的食品** — 測試哪些乳製品對你的肚子更友好。
2. **精於分量** — 適量享用乳製品。
3. **尋找替代的營養源頭** — 尋覓乳製品以外的鈣質和維生素D來源。
4. **隨時準備乳糖分解酵素** — 有準備，就可以隨興享用乳製品。

## 結語：乳糖不耐的生活也可以甜美（無糖版！）

乳糖不耐症在台灣是一個普遍的問題，會影響你的生活方式，但可以不影響你的生活品質。

找到你能承受的乳糖量是很重要的，這樣你就可以享受你喜歡的食物，而不會讓你的腸胃不舒服。
像是一杯拿鐵、兩片起司披薩、或是一小杯冰淇淋。

::alert{type="success"}
做好準備，隨時準備乳糖分解酵素，這樣你就可以想吃就吃。

當然，就算是有乳糖分解酵素，也不要過量食用乳製品，因為乳製品中的脂肪和膽固醇含量都很高，過量食用會增加心血管疾病的風險。

找替代的營養來源來補充鈣質和維生素D，像是綠葉蔬菜、豆類、堅果、魚類、蛋類等等。

了解自己的身體，找到適合自己的生活方式，就可以過著健康又快樂的生活。

通過這些策略，你可以有效地管理乳糖不耐症，改善生活品質。

# 2024-2025年乳糖不耐症診斷與治療最新研究

### 最新診斷方法進展

根據2024-2025年最新研究，乳糖不耐症的診斷有以下重要更新：

**氫氣呼氣測試（黃金標準）：**
- 仍是最佳診斷工具，非侵入性且可靠
- 空腹後攝取乳糖溶液，定期收集呼氣樣本測量氫氣
- 氫氣升高表示未消化乳糖的吸收不良
- 局限性：小腸細菌過度生長或快速轉運時可能出現假陽性

**乳糖耐受血液測試：**
- 測量攝取乳糖後血糖變化
- 血糖上升少於20 mg/dL且伴隨症狀提示乳糖不耐症
- 局限性：糖尿病患者或其他吸收不良症候群患者可靠性較低

**基因檢測：**
- 檢測MCM6基因變異（特別是-13910C/T和-22018G/A）
- 與成人型乳糖酶缺乏（原發性乳糖不耐症）相關
- 優點：有助於區分原發性和繼發性乳糖不耐症
- 局限性：不適用於診斷繼發性乳糖酶缺乏

**飲食排除和症狀追蹤：**
- 測試結果不明確時，建議進行至少4週的乳糖排除試驗
- 配合症狀監測，提供個人化診斷

### 2024-2025年治療與飲食管理策略

**乳糖限制飲食：**
- 一線管理包括減少或消除含乳糖食物（牛奶、軟質起司、冰淇淋）
- 大多數乳糖不耐症患者每天可耐受高達12克乳糖（約一杯牛奶）
- 與其他食物一起攝取時耐受性更好

**酶補充療法：**
- 非處方乳糖酶補充劑可幫助消化乳糖
- 需與乳製品一起服用才能發揮效果

**益生菌和發酵乳製品：**
- 優格和硬質起司通常耐受性更好
- 乳糖含量較低且含有幫助消化乳糖的細菌

**營養諮詢：**
- 預防鈣質、維生素D和其他常見於乳製品的營養素缺乏
- 營養師可能建議強化植物性替代品或補充劑

**個人化營養：**
- 強調基於基因、臨床和生活方式因素的個人化飲食計劃
- 優化症狀控制和營養狀況

### 新興研究方向

**微生物群調節：**
- 研究腸道菌群組成如何影響乳糖耐受性
- 探索針對性益生菌是否能改善症狀

**非侵入性生物標記：**
- 研究新型呼氣或糞便生物標記
- 提供更準確、快速的診斷方法

**數位症狀追蹤：**
- 開發移動應用和數位工具
- 幫助患者追蹤症狀和飲食攝取

#title
什麼是乳糖不耐症？
#description
乳糖不耐症是一種由於乳糖酶基因表達減少，導致身體對含有乳糖的食物容忍度差，從而無法消化乳糖的情況。
::
::card
#title
為什麼亞洲人乳糖不耐症盛行？
#description
乳糖不耐症在亞洲人中盛行是由於遺傳因素和飲食習慣，乳糖酶活性在斷奶後隨年齡減少。
::
::card
#title
乳糖不耐症的症狀有哪些？
#description
症狀包括脹氣、腹脹、打嗝、輕微腹痛，以及由於未消化的乳糖被結腸中特定細菌發酵可能導致的腹瀉和水便。
::
::card
#title
如何診斷乳糖不耐症？
#description
它可以通過觀察食用乳糖後出現的輕微症狀，以及避免乳糖5-7天後的改善來診斷。臨床測試包括乳糖呼氣氫測試和乳糖耐受試驗。
::
::card
#title
對於乳糖不耐症有哪些飲食建議？
#description
主要目標是在維持鈣質攝取的同時避免症狀。這可能涉及限制乳糖攝取，從少量開始逐漸增加，使用消化酶產品，並確保從非乳製品來源或補充劑中獲取足夠的鈣。
::
::card
#title
乳糖吸收不良有哪些不同類型？
#description
類型包括原發性乳糖吸收不良，可細分為先天性乳糖酶缺乏、發育性乳糖酶缺乏，以及通常由腸道疾病引起的次發性乳糖吸收不良。
::
