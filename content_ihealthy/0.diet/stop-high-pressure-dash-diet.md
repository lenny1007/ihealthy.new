# DASH德舒飲食法：停止高血壓的科學飲食方案

## 📋 快速摘要

| 項目 | 詳細資訊 |
|------|----------|
| **主要目的** | 控制高血壓，改善心血管健康 |
| **健康風險等級** | ✅ 低風險 - 安全有效 |
| **開發機構** | 美國國家衛生院（1997年） |
| **評級排名** | 美國新聞最佳飲食第一名 |
| **適用人群** | 高血壓患者，一般健康人群 |
| **核心原則** | 低鈉、高蔬果、全穀物 |

很多人都有**高血壓**的問題，而高血壓是一個很嚴重的疾病，如果不加以控制，會導致心臟病、中風、腎臟病、認知功能障礙等等[^1]。

[得舒飲食(Dietary Approaches to Stop Hypertension，DASH diet)](https://www.nhlbi.nih.gov/education/dash-eating-plan)，意為停止高血壓的飲食型態，由美國國家衛生院國家心肺與血液研究所（NHLBI）於1997年發表的一項飲食方法[^2]。

::alert{type="success"}
德舒飲食法是用來控制高血壓問題的，但是它也是一個很好的健康飲食方法。
::
得舒飲食富含水果、蔬菜、全穀類以及低脂食物；包含魚、肉、家禽、堅果與豆類；

::alert{type="danger"}
並且限制高糖食物及飲料、紅肉以及添加脂肪的攝取。
::

## DASH德舒飲食法的科學基礎

### 研究證據與效果

除了對於血壓的效果之外，得舒飲食也是一個設計給社會大眾的均衡飲食模式。

在台灣，相關研究亦顯示得舒飲食對高血壓初期的病人有降血壓的效果[^3]。

得舒與得舒-鈉飲食的研究結果證明，在富含碳水化合物的飲食模式中強調蔬果與低脂乳製品，並降低飽和脂肪酸、總脂肪與膽固醇攝取，會大大降低血壓與低密度脂蛋白[^4]。

OmniHeart同時也證明若是將一部份的碳水化合物替換成蛋白質(大約一半為植物性來源)或是不飽和脂肪(大部分為單元不飽和脂肪酸)，則可以進一步地降低血壓，低密度脂蛋白膽固醇(LDL-C)與心血管疾病的風險[^5]。

2024-2025年，在美國新聞與世界報導中，得舒飲食持續保持優異排名：2025年獲得「最佳心臟健康飲食」和「最佳高血壓飲食」第一名，「最佳整體飲食」第二名；2024年獲得「最佳整體飲食」和「最佳心臟健康飲食」第二名，「最佳糖尿病飲食」第二名[^6]。

**2024-2025年最新研究發現：**
- **血壓降低效果**：多項最新研究證實DASH飲食能顯著降低收縮壓和舒張壓，效果在數週內即可觀察到[^12]
- **心血管健康改善**：DASH飲食還能改善其他心血管風險因子，如LDL膽固醇和代謝症候群指標[^12]
- **能量限制DASH飲食**：2025年研究顯示，能量限制的DASH飲食在改善心血管風險因子方面優於標準健康飲食[^12]
- **鈉限制的重要性**：DASH飲食與鈉限制結合時效果最佳，建議將鈉攝取量進一步降至1500毫克/天[^12]

得舒飲食的飲食結構與[地中海飲食](/diet/mediterranean-diet)相似。

::alert{type="success"}
富含水果、蔬菜、全穀類以及低脂食物；包含魚肉、家禽、堅果與豆類；
::

## 最新研究：DASH + 限制鈉的飲食法效果更佳

### 研究背景與方法

[研究了什麼？](https://pubmed.ncbi.nlm.nih.gov/37625311/)
單獨或與[地中海飲食](/diet/mediterranean-diet)或降血壓飲食(DASH) 飲食相結合的鹽限制對高血壓患者血壓的影響。

::alert{type="success"}
就算是單獨限制鈉攝取，都有降低血壓的效果，搭配飲食法，效果更好。
::
### 研究結果

結果如何？

與對照組相比，所有 3 個介入組的收縮壓均下降（SRG 限鹽、DDG Dash+限鹽 和 MDG 地中海飲食+限鹽 分別為 –7.6、–11.9 和 –15.1 mmHg）。舒張壓也是如此（SRG、DDG 和 MDG 分別為 –4.4、–6.6 和 –8.1 mmHg）。

與 SRG 相比，DDG 和 MDG 的收縮壓降低幅度更大，MDG 的降低幅度比 DDG 更大。與 SRG 相比，DDG 和 MDG 的舒張壓降低幅度更大，但 MDG 和 DDG 之間沒有差異。

儘管 DDG 和 MDG 之間的收縮壓或舒張壓變化沒有差異，但 24 小時動態血壓變化的結果相似。

::alert{type="success"}
[地中海飲食](/diet/mediterranean-diet)、德舒飲食都是以全榖物、水果、蔬菜、堅果、豆類、低脂乳製品、魚、家禽和橄欖油為主食，並限制紅肉、糖和鹽的攝入。
::
以目前的科學證據來說，這樣的吃法對身體來說，是相對營養健康的。

## DASH德舒飲食法的核心原則

### 飲食選擇的重要性

DASH 德舒飲食法其實對於愛吃肉的人來說，尤其是豬肉、牛肉等紅肉的人來說，是相對痛苦的。

它強調食用蔬菜、水果、全穀物以及瘦肉，同時減少鈉和飽和脂肪的攝取。

但，當你身體都有高血壓時，回過頭來檢視飲食，其實是一件很重要的事情。

### 科學驗證的效果

自1997年引入以來，DASH飲食法經受了嚴格的科學審查。在一項具里程碑意義的干預研究中，459名參與者實踐了DASH飲食法，結果顯示血壓顯著降低——降低程度甚至可媲美服用降壓藥物[^7]。

一項涉及5,545名參與者的關鍵分析顯示，平均收縮壓下降了3.2毫米汞柱，舒張壓下降了2.5毫米汞柱[^8]。這些好處不僅限於高血壓患者，還延伸至不同人群，凸顯了這種飲食法的普遍適用性。

## DASH德舒飲食法與其他飲食法的比較

DASH飲食法與其他流行的飲食法相比有其獨特優勢：

### 與[地中海飲食](/diet/mediterranean-diet)的比較
兩者都強調水果、蔬菜和全穀物，但DASH飲食更注重鈉的限制和低脂乳製品的攝取。

### 與[生酮飲食](/diet/weight-loss-ketogenic-diet)的比較
DASH飲食不限制碳水化合物，而是強調選擇健康的碳水化合物來源，如全穀物和水果。

### 與[間歇性斷食](/diet/intermittent-fasting)的比較
DASH飲食不涉及時間限制，而是專注於食物選擇和營養平衡。

## DASH德舒飲食法的詳細指南

### 食物分類與份量建議

DASH 德舒飲食法是一種包容性的飲食模式，結合了：

::list{type="success"}
- **全穀物**：每天主要為全穀物的6至8份
- **蔬菜**：每天4至5份，是必需礦物質的寶庫
- **水果**：每天4至5份，提供天然甜味和纖維
- **瘦肉蛋白**：適量的瘦肉，並輔以豆類和一些魚類
- **乳製品**：每天2至3份的低脂乳品，提供鈣和蛋白質
- **堅果/種籽**：每周幾份，提供健康脂肪和飽足感
::

### 烹調原則

烹調的方式也很重要，DASH 德舒飲食法相對限制了油脂和鹽的使用，並鼓勵使用香草和香料來增添風味。

### 營養學原理

實際上，透過飲食管理高血壓意味著：

- **全穀物**：提供纖維和必需維他命同時管理胰島素反應
- **蔬菜/水果**：提供豐富的抗氧化劑以及有益心臟的礦物質
- **瘦肉蛋白**：提供必須的氨基酸而不含過多飽和脂肪
- **低脂乳製品**：不增加動脈斑塊的情況下滿足鈣質需求

藉由增加這些食物的攝取並將鈉的攝取量控制在每日2,300毫克以下[^9]，可以有系統地逐步降低血壓。

**2024-2025年最新鈉攝取建議：**
- **標準建議**：每日鈉攝取量控制在2,300毫克以下
- **高血壓患者**：建議進一步降至1,500毫克/天，可獲得更大的血壓降低效果[^12]
- **最佳效果**：DASH飲食與鈉限制結合時，血壓控制效果最佳[^12]

**2024-2025年最新份量建議（基於2000卡路里）：**
- **全穀物**：6-8份
- **蔬菜**：4-5份
- **水果**：4-5份
- **低脂/脫脂乳製品**：2-3份
- **瘦肉/家禽/魚類**：6份或更少
- **油脂**：2-3份
- **鈉**：≤2,300毫克（理想為1,500毫克）[^12]

## 營養學基礎與DASH飲食

了解[巨量營養素](/diet/macronutrients-guide)的平衡對於DASH飲食的成功至關重要。適當的[膳食規劃](/diet/meal-planning-fundamentals)可以幫助你更好地遵循DASH飲食原則。

[腸道健康](/diet/gut-health-fundamentals)也與血壓管理密切相關。良好的腸道菌群可以幫助改善代謝和營養吸收。攝取足夠的[益生菌](/diet/probiotics-prebiotics-guide)和膳食纖維可以促進腸道健康。

## DASH德舒飲食法的風險與注意事項

### 潛在風險

雖然DASH 德舒飲食法的好處範圍廣泛，且得到科學支持，但重要的是要意識到潛在的警示。對於患有特定健康狀況（如晚期腎病）的個體而言，較高的鉀含量可能會造成問題[^10]。

如果自身有慢性病，或者正在服用處方藥物，則應在開始 DASH 德舒飲食法之前與醫生討論。

### 適應期建議

- 逐步調整飲食習慣，避免突然改變
- 配合規律運動和良好生活習慣
- 定期監測血壓變化
- 尋求專業營養師指導

## 血壓之外：DASH對廣泛健康問題的影響

DASH 德舒飲食法對其他健康領域也有正面影響：

### 體重管理
通過優先考慮天然食物和規律餐飲，有助於體重管理。

### 慢性腎病（CKD）
調節蛋白質和電解質攝入可為CKD患者帶來益處。

### 骨骼健康
增加的乳製品攝入，提高鈣和維他命D水平，強化骨骼[^11]。

### 心血管健康
降低膽固醇和血壓，減少心血管疾病風險。

## DASH飲食法實施指南

### 開始DASH飲食的步驟

1. **評估現狀** - 了解當前飲食習慣和血壓狀況
2. **設定目標** - 明確血壓控制目標
3. **逐步調整** - 逐步減少鈉攝取，增加蔬果攝取
4. **監測效果** - 定期測量血壓和體重
5. **持續改善** - 根據效果調整飲食計劃

### 日常飲食建議

**早餐：**
- 全穀物麥片配低脂牛奶
- 新鮮水果
- 堅果或種籽

**午餐：**
- 全穀物三明治配瘦肉
- 大量蔬菜沙拉
- 低脂優格

**晚餐：**
- 烤魚或瘦肉
- 糙米或全穀物
- 蒸煮蔬菜

**點心：**
- 新鮮水果
- 堅果
- 低脂乳製品

## DASH飲食法常見問題解答

### Q1: DASH飲食法需要多久才能看到效果？
A: 通常2-4週內可以看到血壓改善，但建議長期堅持。

### Q2: 可以吃外食嗎？
A: 可以，但要選擇低鈉、低脂的選項，並控制份量。

### Q3: 需要完全戒鹽嗎？
A: 不需要，但要控制在每日2300毫克以下，高血壓患者建議1500毫克以下。美國心臟協會建議理想限制為1500毫克/天。

### Q4: 可以喝酒嗎？
A: 建議限制酒精攝取，男性每日不超過2杯，女性不超過1杯。

### Q5: 運動對DASH飲食法有幫助嗎？
A: 是的，規律運動可以增強DASH飲食法的降壓效果。

## 結語

總而言之，DASH 德舒飲食法作為一種經過細緻研究和驗證的方式，有助於控制高血壓和管理糖尿病。

油脂的攝取量較低，並且鼓勵攝取豐富的蔬菜和水果，這些都是健康的飲食模式。

::alert{type="success"}
必須要有效的控制飲食，養成良好的飲食習慣，才能有效的控制血壓。
::
遺憾的是，好吃、美食、甜食、高熱量的食物，都是很多人的最愛，但是這些食物對於身體的負擔是很大的。

**關鍵要點：**
1. **低鈉飲食**是DASH飲食法的核心
2. **高蔬果攝取**提供豐富的營養素
3. **全穀物選擇**有助於血糖控制
4. **適量蛋白質**維持身體機能
5. **長期堅持**才能看到最佳效果

記住，DASH德舒飲食法不是短期的飲食改變，而是一種長期的健康生活方式。透過科學的飲食原則，我們可以有效控制血壓，改善心血管健康，提升整體生活品質。
