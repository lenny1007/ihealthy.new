# 1歲以下嬰兒不應喝果汁？兒科營養專家解析與安全指南

## 📋 快速摘要

| 項目 | 詳細資訊 |
|------|----------|
| **主要建議** | 1歲以下嬰兒不應喝果汁 |
| **健康風險等級** | ⚠️ 中等風險 - 可能影響發育 |
| **主要問題** | 高糖分、蛀牙風險、營養不足、肥胖風險 |
| **替代方案** | 母乳、配方奶、整個水果 |
| **適用年齡** | 1歲以下嬰兒 |
| **監管機構** | 美國兒科學會（AAP）2024-2025最新指南 |

## ⚠️ 安全指南

::alert{type="warning"}
**重要提醒：**
- 1歲以下嬰兒應避免所有果汁（除非臨床需要）
- 果汁缺乏蛋白質和纖維
- 高糖分可能導致蛀牙和體重問題
- 建議選擇整個水果而非果汁
- 母乳和配方奶是理想營養來源
- 避免在睡前或使用奶瓶/吸管杯長時間飲用果汁
:::

**孩子的健康建立在營養的基石上**，特別是在生命初期的第一年。

渴望給小寶貝們最好的父母們，經常會選擇他們認為是健康的選擇 —— 比如水果汁。但根據美國兒科學會（AAP）2024-2025年最新指南，一歲以下的嬰兒餐單上應該沒有果汁。

這可能看起來與傳統常識不符，AAP的指南揭示了許多原因，為什麼果汁儘管來自自然，可能並不是許多人以為的那樣無害。

## 嬰兒開始吃副食品時，應該避免給他們喝果汁

母乳和嬰兒配方奶一直被認為是一歲以下嬰兒的理想營養來源，為他們提供適合發展的必需營養素混合。

孩子們在探索固體食物時，小零食與方便取得的果汁常常因為孩子們愛吃、愛喝而成為父母們哄孩子的辦法。

但是，美國兒科學會 AAP 2024-2025年最新建議，探索風味和質地的旅程應該遠離果汁，而是鼓勵先從水果開始。

了解[兒童營養健康](/age-specific-health/children-nutrition-health)的重要性，以及[孕期母體健康](/age-specific-health/pregnancy-maternal-health)對嬰兒發育的影響。

## 為什麼水果可以，果汁不行？科學解析

果汁色彩繽紛、甜味十足，很受小朋友歡迎。

然而，**AAP建議嬰兒不應攝取果汁**，原因有很多：

::list{type="primary"}
- **高糖分含量**：果汁含有高量的天然糖分，可能導致攝取過多的卡路里，從而造成不必要的體重增加及對甜食的偏好。即使是100%純果汁也缺乏纖維，無法有效調節血糖波動。
- **蛀牙風險**：尤其是從水瓶或吸管杯喝果汁，由於糖分接觸的時間延長，可能會增加蛀牙的風險。睡前飲用果汁特別危險。
- **營養不足**：果汁缺乏蛋白質和纖維——這些是嬰兒飲食中的關鍵營養素。相比之下，整個水果保留了它們的纖維和更完整的營養素組合。
- **肥胖風險**：2024-2025年最新研究顯示，6個月前就開始喝果汁的嬰兒，在兒童期更容易攝取含糖飲料，BMI指數也較高。
- **營養密度食物被取代**：果汁可能取代更有營養的飲品（母乳、配方奶、牛奶、水）和食物，導致營養不足。
- **可能的藥物干擾**：果汁還可能干擾某些藥物的吸收。
:::

::alert{type="success"}
整個水果保留了它們的纖維和更完整的營養素組合，而果汁失去了纖維後容易攝取過量的糖分。2024-2025年研究證實，早期果汁攝取與後期肥胖風險增加相關。
:::

這些指南呈現了一種思維轉變，提醒我們現在重新評估曾經普遍的飲食習慣。

參考[營養素基礎指南](/diet/macronutrients-guide)了解蛋白質和纖維的重要性，以及[閱讀營養標籤](/diet/reading-nutrition-labels)如何識別糖分含量。

## 回顧過去：為什麼果汁曾經被接納

果汁過去因為含有維他命C並附帶額外的補水好處而受到推崇。

它也被視為對便秘有幫助的一種飲品，更不用說它因為又美味又營養而流行。

然而，2024-2025年的研究發現，這些因素可能不足以抵消潛在的不利面。

當涉及嬰兒和果汁時，負面因素似乎超越了正面因素。

糖分過多可能導致牙齒問題和超重狀況，加上它缺乏必要的膳食纖維和蛋白質，使得整個水果在營養競賽中成為明顯的勝利者。

缺乏這些基本成分的果汁，簡直不是我們曾以為的英雄。

::alert{type="warning"}
過去人們說孩子胖不是真的胖，但2024-2025年研究發現，小時候胖會更容易長大後有肥胖的問題，早期果汁攝取是重要風險因素[^1][^2]。
:::

了解[減重飲食指南](/diet/diet-loss-weight)和[避免假期增重](/diet/weight-gain-avoid-in-holidays)的重要性，以及[維生素C補充](/food-supplement/vitamin-c)的更好來源。

## 那麼對於大一點的孩子和成人的果汁建議呢？

對於一歲以上的兒童，仍須謹慎處理果汁。

**2024-2025年AAP最新建議限制攝取量：**
- **幼兒（1-3歲）**：每天最多4盎司（120毫升）
- **兒童（4-6歲）**：每天4-6盎司（120-180毫升）
- **兒童和青少年（7-18歲）**：每天最多8盎司（240毫升）

**重要注意事項：**
- 不要用奶瓶或吸管杯提供果汁，避免整天飲用
- 不要在睡前提供果汁
- 鼓勵食用整個水果而非果汁
- 避免未經巴氏殺菌的果汁產品

這些限制反映了即使孩子們在成長，適量飲用果汁仍然是關鍵。

但成人呢？他們也不應讓果汁溢出他們的杯子。

建議每天不超過240毫升果汁，因為整個水果給予了他們更豐富的纖維和營養物[^4]。

::alert{type="warning"}
小朋友大約就是一罐養樂多的量，而大人就是兩罐多些。大一點的馬克杯就超過這個量了，所以要注意。
:::

參考[地中海飲食指南](/diet/mediterranean-diet)了解健康飲食模式，以及[膳食規劃基礎](/diet/meal-planning-fundamentals)如何為兒童準備營養均衡的餐點。

## 預防策略和實用建議

基於2024-2025年最新研究，以下是有效的預防策略：

::list{type="success"}
- **推廣水和牛奶**：斷奶後，水和低脂或脫脂牛奶是兒童的主要飲品選擇
- **教育家庭**：教導父母和照顧者關於整個水果的好處和過度飲用果汁的風險
- **限制接觸**：避免讓果汁隨時可得，不要用它來安撫哭鬧的孩子或作為獎勵
- **監控攝取**：兒科醫生應在健康檢查時篩查果汁和含糖飲料攝取量，並相應地諮詢家庭
- **文化敏感指導**：研究顯示不同種族和民族的果汁消費模式有所不同，需要考慮文化因素
:::

## 對於果汁和嬰兒的結論思考

當我們考慮全局，很明顯，AAP的建議不僅僅是一項指令，更是關於構成一個孩子身體的營養基石是什麼的對話開端。

水果很好，只是果汁不是。

就算是大人，目前主流的飲食建議也是盡量吃原型食物，而不是液態的食物。

液態的食物如牛奶、果汁、豆漿等，雖然營養豐富，但一不小心容易攝取過量。

因此，當我們尋找營養選項時，**讓我們在購物籃裡為整個水果騰出空間，將果汁留在貨架上** —— 至少直到孩子慶祝他的第一個生日。

小時候的營養習慣，會影響到長大後的健康。

而不給孩子果汁，也是一種營養教育。

2024-2025年最新研究顯示給兒童喝果汁的缺點，例如高糖分導致過量熱量攝取、增加蛀牙風險、缺乏蛋白質和纖維、可能的異常體重增加、與藥物吸收的互動、營養密度食物被取代，以及早期果汁攝取與後期肥胖風險增加相關。

#title
根據2024-2025年新指南，成人應如何消費果汁？
#description
健康的成人不需要喝果汁，如果他們喝，應該不超過兒童的最大限量，即每天240毫升。由於保留了膳食纖維和營養素，因此建議食用整個水果。避免在睡前飲用，不要用吸管杯長時間飲用。
