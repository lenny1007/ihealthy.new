# 如何選擇防曬乳？完整防曬指南與SPF、PA等級解析

## 📋 快速摘要

| 項目 | 詳細資訊 |
|------|----------|
| **主要問題** | 防曬乳選擇與使用指南 |
| **健康風險等級** | ⚠️ 中等風險 - 需正確使用 |
| **核心原則** | 廣譜防護，正確塗抹，定期補塗 |
| **SPF選擇建議** | 日常使用：SPF 30-50，戶外活動：SPF 50+ |
| **PA等級選擇** | PA+++以上，加號越多防護越強 |
| **塗抹量標準** | 臉部：十元硬幣大小，全身：約30ml |
| **塗抹時機** | 外出前15-30分鐘，每2小時重新塗抹 |
| **防曬類型** | 物理防曬（溫和）vs 化學防曬（輕薄） |

## ⚠️ 安全指南

::alert{type="warning"}
**重要提醒：**
- 防曬乳不能完全阻擋紫外線，應配合其他防護措施
- 如有皮膚過敏或不適，請立即停用並諮詢醫師
- 嬰兒6個月以下避免使用防曬乳
- 避免接觸眼睛，如有接觸請立即沖洗
- 配合帽子、太陽眼鏡、防曬衣物等防護措施
- 避免高溫環境存放，開封後12個月內使用
- **2024-2025年FDA建議：** 優先選擇GRASE認證的礦物防曬成分（氧化鋅、二氧化鈦）
- **2024-2025年新規定：** 防曬乳必須標示有效期限，未標示者應在購買後3年內使用
- **2024-2025年安全提醒：** 化學防曬成分仍在FDA審查中，建議遵循使用說明並避免過度使用
::

# 2024-2025年防曬乳法規與成分更新

### 最新FDA法規變更

根據2024-2025年美國食品藥物管理局（FDA）的最新規定，防曬乳成分和標示有以下重要更新：

**GRASE認證成分（2024-2025年）：**
- 目前僅有氧化鋅（Zinc Oxide）和二氧化鈦（Titanium Dioxide）被FDA認定為「一般公認安全有效」（GRASE）
- 這兩個礦物防曬成分可同時防護UVA和UVB射線
- 其他12種化學防曬成分仍在FDA安全審查中

**化學防曬成分審查（2024-2025年）：**
- 包括阿伏苯宗（Avobenzone）、氧苯酮（Oxybenzone）、辛基三嗪酮（Octocrylene）等成分仍在審查中
- FDA要求提供更多關於系統性吸收和潛在健康風險的數據
- PABA和托拉明水楊酸鹽已被認定為不安全，禁止在美國防曬乳中使用

**新成分發展（2024-2025年）：**
- 貝莫三嗪醇（Bemotrizinol, Tinosorb S）正在FDA審查中，預計2026年獲批
- 這將是25年來首個新增的防曬活性成分
- 提供強效的UVA和UVB防護，具有高光穩定性和更好的化妝品效果

**SAFE防曬標準法案（2024年）：**
- 旨在推動FDA現代化和澄清防曬成分評估
- 可能加速新防曬成分的批准流程
- 促進全球認可的防曬成分在美國市場的應用

### 消費者權益保障

這些法規更新旨在：
- 提高防曬乳成分的安全性和透明度
- 加強消費者對防曬產品安全的信心
- 促進防曬科技的創新和發展
- 與國際防曬標準接軌

# 為何我們需要防曬？

陽光雖然是維持生命所必需的，但就像許多事物一樣，過猶不及。

皮膚，作為我們身體的最大器官，是對抗包括陽光中強烈紫外線（UV）等各種環境因素的保護屏障。

長期過度曝曬這些光線會導致皮膚損傷，從輕微的曬傷到嚴重的皮膚癌。

如果只是看起來老化也就算了，過度UV暴露導致的DNA干擾顯著提高了多種形式的皮膚癌風險，例如基底細胞癌、鱗狀細胞癌和可能致命的黑色素瘤（La Roche-Posay, n.d.; Mayo Clinic, n.d.）。

## 防曬乳在防什麼? UVA 和 UVB

防曬乳不僅是為了避免曬傷；它也是我們對抗紫外線（UV）輻射的第一道防線。

陽光包含不同類型的光線，以 UVA 和 UVB 為皮膚健康的主要關注點。

UVB是短波輻射，只影響表皮或皮膚表面，導致發紅和曬傷。

相對地，長波的UVA光線更深入皮膚層，造成曬黑、皮膚老化和DNA損傷。

盡管UVA和UVB都會加速皮膚癌和早衰的風險，過度的角質化、皺紋和囊腫樣痤瘡也與UVA曝曬相關（Cleveland Clinic, n.d.; Verywell Health, n.d.）。

在UV光譜中的兩個主要反派是：

::list{type="primary"}
- **UVB:** 導致曬傷的罪魁禍首，以及皮膚癌發展的邪惡角色。
- **UVA:** 背後操控曬黑、皮膚老化的隱形者，也是皮膚癌風險的幫兇。

一款好的防曬乳會提供對 UVA 和 UVB 的保護 —— 被標記為「廣譜」—— 因為兩者都能以不同但同樣令人擔憂的方式傷害皮膚[1]。

## 防曬乳的 SPF 防曬係數、UVA 保護和 PA 防曬等級代表什麼?

防曬係數，也就是SPF，指示一款防曬霜抵禦UVB導致皮膚曬傷的能力。

數學上，SPF是通過比較受保護與未保護的皮膚發生曬傷的時間來衡量。

例如，如果未保護的皮膚在10分鐘後變紅，那麼理論上，SPF30的防曬霜允許你在陽光下停留30倍時間，即300分鐘，而不會曬傷[^3][^4]。

但是，別忘記，這是在理想條件下的情形，還需考慮皮膚類型、陽光強度以及塗抹厚度等重要因素。

防曬指數（SPF）是衡量防曬乳抵禦UVB射線能力的數字等級。

想要在不受傷害的情況下把皮膚曬成古銅色嗎？ SPF等級是你的指南。

高SPF提供更長久的保護，但重點來了：沒有防曬霜能提供100%的阻隔[2]。

一般的建議是你需要的是寬譜SPF30或更高，這能過濾掉大量的UVB輻射，使你在享受陽光的同時不用擔心皮膚受損。

::alert{type="success"}
請選擇寬譜SPF 30或更高的防曬霜。

除了 SPF 防曬係數外，你看到的 UVA 保護和 PA 防曬等級 是什麼?

雖然SPF評估對UVB的保護，但它並不衡量對UVA的保護程度。

這時PA等級，即UVA防護等級，就派上用場了。

這個源自日本的等級，將UVA保護水平簡化為"+"系統，從PA+（一些UVA保護）到PA++++（極高的UVA保護）。

加號越多，UVA防禦力就越強[^2][^5]。

::alert{type="success"}
由於 SPF 僅僅針對 UVB，因此 PA 等級是衡量防曬乳對 UVA 保護的指標。 PA +越多，表示防曬乳對 UVA 的保護越好。

所以除了 SPF，你還需要 PA 等級來衡量防曬乳的 UVA 保護。

廣譜防曬乳，「廣譜」意味著防曬霜能對抗UVA和UVB全範圍的光線，這點至關重要，因為這兩種光線都會對皮膚造成傷害。

美國食品藥物管理局（FDA）建議使用廣譜型防曬霜，並強調至少選擇SPF15以對抗曬傷、皮膚癌和提前衰老[^1]。

一般防曬乳有兩種不同類型的成分 : 物理防曬的礦物基底成分與化學成分為主的防曬基底。

物理防曬的礦物防曬乳，是比較早期的防曬乳，主要成分是以「氧化鋅」或「二氧化鈦」，透過反射，將UV射線反射掉，這兩個成分不會導致珊瑚白化，因此對海洋的影響也較小。

雖然說礦物防曬乳的成分不會導致珊瑚白化，但缺點是因為礦物防曬乳的成分較大，所以在塗抹的時候會有一層很明顯的白色上在你的皮膚上，這是礦物防曬乳的缺點。

而化學防曬霜則是讓UV射線進入，然後將它們轉變為無害的溫暖光線。

化學性防曬乳中常見的成分氧苯酮（Oxybenzone）或甲氧基肉桂酸辛酯（Octinoxate），化學性的防曬乳在使用的時候更為輕薄透明，化妝品如果有防曬效果的話，大多是化學性的防曬乳。 

因為很少人喜歡防曬乳塗完臉後，白白一片的。

::alert{type="success"}
化學性防曬乳在防曬性能測試中表現就為優秀，特別是那些有著高SPF評級的產品。

::nuxt-image{imgSrc="/med_images/optimized/med_files_129.webp" style="max-width: 100%; width: 600px; height: auto; display: block; margin: 1rem auto;" width="600" height="400" preset="card" altText="Health information: med files 129"}
::

::badge{type="info"}
礁石安全防曬
:: 
::badge{type="info"}
珊瑚礁保護
:: 
::badge{type="info"}
防曬環境影響

但是化學性防曬乳因為成分較小，所以會被珊瑚吸收，導致珊瑚白化，也由於分子較小，也可能會被人體的系統吸收，而有安全性疑慮。

物理性防曬霜較不可能被吸收，普遍認為更安全[^1][^7]。

::alert{type="success"}
如果你的肌膚是很敏感的，或是你會下海，盡量選擇「非奈米」或「礦物基底」的防曬產品。
需要化妝的話，可以選擇化學性的防曬乳。
一般情況兩種都沒差。

在挑選礦物基底的防曬產品時，可以檢查包裝上有無非奈米的字樣。「非奈米」的意思是防曬產品內的粒子超過100奈米，也因為粒子較大，所以不會被珊瑚吸收。

## 防曬乳多久要重複塗抹一次? 流汗會影響防曬嗎？

正確地使用防曬乳與選擇合適的防曬霜同等重要。

::nuxt-image{imgSrc="/med_images/optimized/med_files_86.webp" style="max-width: 100%; width: 600px; height: auto; display: block; margin: 1rem auto;" width="600" height="400" preset="card" altText="Health information: med files 86"}
::

::badge{type="info"}

:: 
::badge{type="info"}
紫外線攝影機防曬測試
:: 
::badge{type="info"}
立即防曬保護
:: 

建議的防曬乳塗抹量為每平方公分2毫克，大約是整個臉部一枚十元硬幣的大小。
在外出前15至30分鐘，將防曬霜充足地塗抹於所有暴露的皮膚上，並至少每兩小時重新塗抹一次。

然而，流汗可能會將產品厚度減半，這會導致SPF防護指數從50 顯著下降到大約7.1。

測試顯示，對於戶外活動時流汗，建議每一到兩小時重新塗抹一次；
而對於室內較少活動的情況，一天過半時重新塗抹一次可能就足夠了。

大量出汗或游泳後需要更頻繁地重新塗抹，無視任何稱為「耐汗」或「防水」的聲稱，因沒有任何防曬霜是完全抵禦水的[^3][^4]。

::list{type="primary"}
- **提前且大量塗抹：** 在你的皮膚與陽光來個約會前大約15-30分鐘擦上防曬品[^8]。
- **分量要足：** 全身需要一盎司，或足夠裝滿一個小酒杯的量。臉部則以十元硬幣大小的一團為准[^9]。
- **包括每個角落：** 不忽略頸部、耳朵，甚至是腳背。這些容易被忘記的地帶是日曬損傷的重災區。
- **養成習慣：** 將防曬納入你的日常保養，確保你從紫外線的襲擊中一直處於保護之下[^10]。

一般建議的外出前15分鐘使用防曬乳，主要是針對化學性的防曬乳。
物理性的防曬乳，因為是反射的方式，所以不需要等待，抹上去就有防曬效果。

::alert{type="warning"}
防曬乳抹太少的話，防護時間會降得很短，建議會曬到的地方都抹多點。

## 額外的防曬策略 - 防曬衣、帽子和太陽眼鏡

上面有提到，不管是流汗還是玩水，都會導致防曬乳的效果減半。

所以現在也有人在去浮潛或是去海邊玩水時，會穿上防曬衣，這樣就不用擔心防曬乳的效果會減半。

防曬衣的防曬效果是由UPF來衡量的，UPF是衡量防曬衣防曬效果的指標。

想像UPF是面料的SPF（防曬指數）。

UPF等級制度準確量化衣服允許多少UV輻射穿透到你的皮膚。

若衣物有UPF50的標籤，僅有1/50的太陽UV光線能透過材料，相當於為你的肌膚提供了驚人的98%輻射阻隔[^1^]。

::list{type="primary"}
- **UPF 15-24**：良好的UV防護
- **UPF 25-39**：非常好的UV防護
- **UPF 40-50+**：極佳的UV防護

一般 UPF 30 的衣物已能提供滿意保護，但若長時間在陽光下曬，UPF 50+會是更好的選擇。

不是所有的面料都能同等阻擋UV光線。合成材質如聚酯纖維在這方面表現優異，而某些如緊密棉也頗為不錯。
編織或針織越緊密，透過的UV光線就越少。簡單來說，你與太陽中間的布料越多越好。

深色調吸收更多UV光線，提供比淺色系更高的防護。

當然，除了防曬衣外，帽子、太陽眼鏡也是防曬的好幫手。

## 防曬的太陽眼鏡

其實在做戶外運動時，除了防曬衣外，防曬的太陽眼鏡也是很重要的。
尤其是長時間需要曝曬在陽光下的戶外活動，像是跑馬拉松、三鐵等。

參考[運動類型與益處](/personal-health/exercise-types-benefits)了解不同戶外運動的防曬需求，以及[跑步膝蓋保護](/personal-health/running-hurt-knee)中關於戶外運動防護的建議。

防曬乳塗抹不到眼睛，也無法防止眼睛受到紫外線的傷害。 紫外線（UV）輻射對我們的眼睛造成嚴重的風險。
例如黃斑部病變、角膜損害，甚至眼睛癌症等病症皆可追溯至紫外線照射。

UV400標準的太陽眼鏡是您的最佳選擇，確保防護至400納米波長[^1]。
一般開車時，如果通勤的路徑有經過下午夕陽西曬的公路的話，也建議戴上太陽鏡，避免眼睛受到紫外線的傷害。

可以視你的需要，選擇不同的太陽眼鏡，像是運動太陽眼鏡、開車太陽眼鏡、或是時尚的太陽眼鏡。

另外就是有一種會隨著陽光變色的變色鏡片又稱為「全視線鏡片」，當鏡片受到紫外線照射時，鏡片顏色便會在短時間內變深，從而達到阻隔紫外線的功效。 

然而，部份全視線鏡片並不具備過濾反射性紫外線的功能，如果需要長時間進行戶外活動，還是建議選配太陽眼鏡有效地保護雙眼。

::alert{type="warning"}
並不是顏色越深的鏡片，防護效果就越好，因為鏡片顏色深淺與防護效果並無直接關係，而是取決於鏡片材質。

## 有防曬效果的化妝品

一般來說，有防曬效果的化妝品，都是化學性的防曬乳，所以如果你的皮膚對化學性的防曬乳有過敏的話，建議還是不要使用有防曬效果的化妝品。

而且化妝品的防曬效果，也是有限的，所以如果你需要長時間曝曬在陽光下的話，還是建議使用防曬乳。

::alert{type="warning"}
化妝品的防曬效果就當作輔助，不要當作主要的防曬產品。

其實化妝品的防曬效果不佳，最主要的原因是因為大家都不會塗抹足夠的量，所以才會導致防曬效果不佳。

## 結論：明智防曬

防曬乳是你的無形盔甲，你看不見的護膚英雄，現在，你可以選擇你的喜歡的防曬乳了。

::alert{type="warning"}
最好的防曬乳是已經塗抹在身上的防曬乳。

了解SPF數字的數字代表的是可以防曬傷的時間，而不是強度。
PA等級是衡量防曬乳對 UVA 保護的指標。

防曬乳的成分，有物理性的礦物防曬乳和化學性的防曬乳，礦物防曬乳的成分較大，所以會有白色的殘留物，而化學性的防曬乳則是透明的。

陽光可以讓你身體產生維生素D，也會讓心情變得比較好，但是過度曝曬會導致皮膚老化和皮膚癌。

好好地曬太陽，但是要記得穿上你的防曬乳。這是迎接陽光最聰明的方式。

了解[維生素D補充](/food-supplement/vitamin-d)的重要性，以及如何在防曬和獲得足夠維生素D之間取得平衡。對於經常進行[戶外運動](/personal-health/exercise-types-benefits)的人來說，正確的防曬策略尤為重要。

#title
什麼是UVA和UVB？
#description
UVA和UVB代表兩種不同波長範圍的紫外線。UVB主要引起曬傷，而UVA主要負責曬黑和老化。防曬保護應根據個人目標選擇，較長波長（UVA）比UVB更難阻擋。
::
::card
#title
SPF和PA代表什麼？
#description
SPF表示延長曬傷發生前的時間的能力，而PA表示延遲曬黑的能力。SPF用數字表示，而PA用加號表示。較高的SPF數字意味著延長曬傷發生前的時間的能力更強，PA中的加號越多，表示延遲曬黑的能力越好。
::
::card
#title
什麼是關鍵波長？
#description
關鍵波長測量防曬產品在不同紫外線波長下的保護「廣度」。更大的廣度意味著更全面的保護。關鍵波長值可以看作產品能夠保護的最大波長。
::
::card
#title
星級評分是什麼意思？
#description
防曬產品上的星級評分強調UVA和UVB之間保護的「均勻性」。更高的均勻性意味著產品提供更平衡完整的保護。
::
::card
#title
什麼是物理和化學防曬？
#description
物理防曬通過反射或散射紫外線來預防皮膚損傷，而化學防曬吸收紫外線並將其轉化為熱能。物理防曬刺激性較小，但可能會顯得白皙和油膩，儘管納米級產品正在改善這些缺點。化學防曬更透明，但可能存在環境或健康方面的顧慮，這些顧慮正在持續監控以確保安全。
::
::card
#title
如何根據個人需求選擇防曬產品？
#description
選擇合適的防曬產品需要考慮個人的年齡、膚質和生活方式。例如，嬰兒應避免使用防曬產品，而推薦1-6歲兒童使用不含化學成分的物理防曬。從6歲開始，除非有皮膚不適，否則可以使用任何防曬產品。對於不同的膚質，如油性、容易長痘或敏感肌膚，建議使用特定類型的產品。
::
::card
#title
足夠的防曬量是多少？
#description
根據FDA和皮膚科協會的說法，每平方厘米皮膚2毫克的防曬劑被認為是足夠的，這相當於整個臉部一個十元硬幣大小，或手掌大小區域的一顆黃豆大小。如果使用的量少於所需量，例如SPF 50產品的一半厚度，則不會得到SPF 25的保護，而是降低到大約SPF 7.1。
::
::card
#title
推薦使用防曬噴霧嗎？
#description
由於分布不均和安全顧慮（如易燃性及眼睛或肺部暴露的風險），不建議使用防曬噴霧。
::
::card
#title
是否需要在室內或晚上擦防曬？
#description
如果坐在紫外線能穿透的玻璃附近，或者在一天中的某個時候會暴露在陽光下，則需要在室內擦防曬。晚上通常不需要，除非有戶外暴露。
::
::card
#title
出汗時如何重新塗抹防曬？
#description
如果流汗，建議每一到兩小時重新塗抹防曬。如果沒有化妝，可以用紙巾擦掉汗水並重新塗抹。如果化妝了，在不破壞妝容的情況下重新塗抹是挑戰性的，而且防曬粉可能不提供強力保護。
::
::card
#title
如何讓防曬產品對珊瑚和海洋友好？
#description
關於防曬導致珊瑚漂白的證據並不非常清晰。被認為對環境風險較高的成分包括具有高生物累積性、使用量、暴露濃度或毒性的成分，如辛基甲氧基肉桂酸酯、4-甲基苯基亞甲基樟腦和苯酮-3。避免這些成分可以減少潛在的傷害。
::
::card
#title
是否應該用防曬保護頭皮？
#description
對於頭髮豐富的人來說，頭髮提供了基本的物理防曬保護。然而，禿頭的人或那些暴露於非常強烈陽光下的人應該考慮在頭皮上使用防曬產品或戴具有防曬功能的帽子。
::
::card
#title
防曬的 ABC 原則是什麼？
#description
防曬的 ABC 原則代表避免（避免在上午10點到下午2點之間的戶外活動）、阻擋（使用適當SPF的防曬產品並每2小時重新塗抹一次）和覆蓋（穿著防護性衣物，如長袖、帽子和太陽鏡）。
::
